# Initial Role Model

## Platform Roles

- `platform_admin`
- `support_admin`

## Tenant Roles

- `tenant_owner`
- `tenant_manager`
- `recruiter`
- `sales_rep`
- `finance_manager`

## Notes

- platform roles never bleed into tenant data ownership
- tenant users may have multiple roles within the same tenant
- payout approval should be limited to `tenant_owner` and `finance_manager`
